# Plane App Testing with Playwright

## About

The project folder has the following structure:

```
- tests
  - api
  - data
  - setup
  - ui
    - fixtures 
    - pages
    - specs
    - utils  
```

## Installation

Change directory:

```shell
cd tests/e2e-playwright
```

Install dependencies:

```shell
npm -i 
```

## Run tests

```
npx npx playwright test
```
