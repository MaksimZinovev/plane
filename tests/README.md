# Plane App Testing

## About

The project folder has the following structure:

```- tests
  - api
  - data
  - setup
  - ui
    - fixtures 
    - pages
    - specs
    - utils  
```

## Installation

Change directory:

```
cd tests/e2e-playwright
```

Install dependencies:

```
npm -i 
```

## Run tests

```
npx npx playwright test
```
